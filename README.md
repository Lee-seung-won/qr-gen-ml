# qr-gen-web

중간 과제용 QR 생성기 웹 서비스 저장소입니다.

## 목표

- URL 또는 텍스트를 입력받아 QR 코드를 PNG로 반환
- `/health` 엔드포인트로 서비스 상태 확인
- GitHub Actions, Docker, Render까지 이어지는 DevOps 파이프라인 구성
